### Core Concepts

- Average = Sum of observations / Number of observations
- Sum = Average × Number of observations

### Important Formulas

- **Average of first n natural numbers** = (n+1)/2
- **Average of first n even numbers** = (n+1)
- **Average of first n odd numbers** = n
- **Average of consecutive numbers** = (First + Last)/2
- **Average of squares of first n natural numbers** = [n(n+1)(2n+1)]/6n

### Weighted Average

When quantities with different averages are combined: Average = (n₁A₁ + n₂A₂ + ... + nₖAₖ)/(n₁ + n₂ + ... + nₖ)

### Key Concepts

- When a new element joins, Average change = (New element - Old average)/(New count)
- When an element leaves, Average change = (Old average - Element)/(New count)

### Practice Problems

**Example 1**: Average of 5 numbers is 40. If one number (50) is replaced by 70, what's the new average? **Solution**: Sum increases by (70-50) = 20. New average = 40 + 20/5 = 44

**Example 2**: Average age of 30 students is 15 years. If teacher's age (45 years) is included, what's new average? **Solution**: New average = (30×15 + 45)/31 = 495/31 ≈ 15.97 years
